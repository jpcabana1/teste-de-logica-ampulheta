# Fase 1 - Desafio de Lógica

## Cálculo da Ampulheta

### Introdução 

Dada uma ampulheta desenhada no console:

```
####################
####################
# ################ #
#  ##############  #
#   ############   #
#    ##########    #
#     ########     #
#      ######      #
#       ####       #
#        ##        #
#        ##        #
#       #  #       #
#      #    #      #
#     #      #     #
#    #        #    #
#   #          #   #
#  #            #  #
# #              # #
##                ##
####################
n = 20
```
O programa deverá desenhar uma ampulheta com n >= 20, onde `n` é o input dado ao usuário.
